— Step 02
Pick a name that
describes the tool

02 / 07
— Step 03
Define
your tool
Open tools/get_weather.ts and replace the stub with a typed tool definition.
tools/get_weather.ts
import { tool, z } from '@anthropic/mcp-cli';

export const get_weather = tool({
  name: 'get_weather',
  description: 'Get current weather for a city',
  input_schema: z.object({
    city: z.string()
             .describe('City name, e.g. Tokyo'),
  }),
  async run({ city }) {
    return fetchWeather(city);
  },
});
03 / 07
— Step 04
Relative path
from project root
Wire it
to Claude
Add the server to .claude/mcp.json under the mcpServers key with your stub's path.
.claude/mcp.json 
{
  "mcpServers": {
    "weather": {
      "command": "node",
      "args": ["weather-server/index.js"]
    }
  }
}

— Step 05

Restart Claude Code. Ask: "what's the weather in Tokyo?" — the tool should fire automatically.
Quit and relaunch Claude Code completely
Verify mcp.json is saved in your project root
Ask Claude to call get_weather with your city
